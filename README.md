# antMobile
ant-design-mobile安装及测试/n

git克隆到本地
git clone https://github.com/nangongxixi/antMobile.git

cd antMobile  进入目录

npm i    安装依赖

npm run dev  或者  npm start    运行

http://localhost:8000/#/test   可访问测试地址
